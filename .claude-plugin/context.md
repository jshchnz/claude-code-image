# claude-code-image Plugin Context

This plugin enables AI image generation and editing. Claude should use this plugin when users request image creation or editing.

## When to Activate

### Image Generation
Recognize requests like:
- "create/generate/make an image of..."
- "draw/illustrate/design..."
- "I need a picture/image/illustration of..."
- "can you make me a..."
- "visualize..."
- Any request for creating visual content (logos, icons, art, photos, etc.)

### Image Editing
Recognize requests like:
- "edit/modify/change this image..."
- "add/remove something from this image..."
- "make this image look like..."
- "apply [style] to this image..."
- Any request to alter an existing image

## Available Commands
- generate.md - Create images from text descriptions
- edit.md - Edit existing images
- configure.md - Set up API keys and preferences
- models.md - List available AI models
- history.md - View past generations
- status.md - Check configuration status
